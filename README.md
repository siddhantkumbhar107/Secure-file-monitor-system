# 🔐 Secure File Transfer Monitoring System

> 🚀 A real-time cybersecurity system to detect unauthorized file transfers, data leakage, and malware activity.

---

## 📘 Description

The Secure File Transfer Monitoring System monitors file activities in real time, detects unauthorized file transfers, verifies file integrity using hashing, and generates alerts for suspicious actions such as data leakage, malware activity, and unauthorized access.

---

## 🚀 Features

- Real-time file monitoring (create, modify, delete)
- Sensitive file detection
- File integrity verification (SHA-256)
- Unauthorized user detection
- USB device monitoring
- Alert generation system
- Activity logging

---

## 🛠 Technologies Used

- Python  
- watchdog  
- hashlib  
- psutil  
- threading
